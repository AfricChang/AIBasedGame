# 水果连连看 (Fruit Link Game)

这是一款经典的水果主题连连看游戏，使用原生HTML、CSS和JavaScript实现。游戏界面美观，交互流畅，并包含计时、计分、提示和洗牌等核心功能。

## 核心功能

- **经典玩法**: 寻找并连接两个相同的可消除水果图标。
- **连接规则**: 两个图标之间的连接路径不能超过两个拐点。
- **动态计分**: 每成功消除一对水果，获得10分。
- **倒计时**: 游戏总时长为120秒，时间耗尽则游戏结束。
- **辅助功能**:
  - **提示**: 消耗5分，高亮显示一对可以消除的水果。
  - **洗牌**: 消耗3分，将当前棋盘上剩余的水果重新排列。
- **可视化连接**: 成功连接时，会在两个水果之间绘制出清晰的连接线。
- **胜利反馈**: 成功消除所有水果后，会弹出胜利界面并显示最终得分。
- **响应式设计**: 游戏界面能适应不同尺寸的设备屏幕。

## 如何游戏

1.  **开始游戏**: 页面加载后游戏自动开始，计时器启动。
2.  **选择水果**: 点击一个水果图标，它将被高亮选中。
3.  **进行配对**: 再点击另一个相同的水果图标。
4.  **成功消除**: 如果两个水果可以根据规则连接，它们将被消除，同时得分增加，连接路径会短暂显示。
5.  **消除失败**: 如果两个水果无法连接，或者选择了不同的水果，选择状态会更新至最新点击的水果上。
6.  **使用道具**: 在遇到困难时，可以点击“提示”或“洗牌”按钮来帮助游戏继续进行。
7.  **游戏结束**: 在规定时间内消除所有水果即为胜利；若时间用完仍有剩余水果，则游戏结束。

## 技术实现

- **前端技术**: 
  - `HTML`: 构建游戏的基本结构。
  - `CSS`: 负责游戏界面的样式、动画和响应式布局。
  - `JavaScript (ES6)`: 实现所有游戏逻辑，包括：
    - 棋盘生成与随机化 (Fisher-Yates洗牌算法)。
    - 用户交互（点击、选择）。
    - 核心消除逻辑（路径查找算法）。
    - 计分、计时和道具功能。
- **Canvas API**: 用于在棋盘上动态绘制水果之间的连接线，提供直观的视觉反馈。
